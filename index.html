Task a) Compute the transmission delay and propagation delay for both options

Transmission Delay: Time to push all data bits onto the transmission medium

Propagation Delay: Time for signal to travel through the physical medium

Formulas:

Transmission Delay = Frame Size / Transmission Rate
Propagation Delay = Distance / Propagation Speed

Option 1: Fiber Optic (1 Gbps)

Transmission Delay:

Frame size = 1500 bytes = 12,000 bits

Transmission rate = 1 Gbps = 1,000,000,000 bps

Transmission Delay = 12,000 ÷ 1,000,000,000 = 12 μs

Propagation Delay:

Distance = 5 km = 5,000 m

Propagation speed = 2 × 10⁸ m/s

Propagation Delay = 5,000 ÷ (2 × 10⁸) = 25 μs

Option 2: Wireless (200 Mbps)

Transmission Delay:

Frame size = 1500 bytes = 12,000 bits

Transmission rate = 200 Mbps = 200,000,000 bps

Transmission Delay = 12,000 ÷ 200,000,000 = 60 μs

Propagation Delay:

Distance = 5 km = 5,000 m

Propagation speed = 3 × 10⁸ m/s

Propagation Delay = 5,000 ÷ (3 × 10⁸) = 16.7 μs

Answer: Fiber has lower transmission delay (12μs vs 60μs) but higher propagation delay (25μs vs 16.7μs) compared to wireless.

Task b) Calculate the link utilization (efficiency) for each case using window size N = 10

Link Utilization: How efficiently the connection can be used with Go-Back-N ARQ protocol

Formula:

Utilization = min(1, N / (1 + 2a))
where a = Propagation Delay / Transmission Delay

Option 1: Fiber Optic Analysis

Step 1: Calculate 'a' parameter

a = Propagation Delay / Transmission Delay = 25 / 12 = 2.08

Step 2: Calculate utilization

Utilization = min(1, 10 / (1 + 2×2.08)) = min(1, 10 / 5.16) = min(1, 1.94)

Result: 100% utilization

Option 2: Wireless Analysis

Step 1: Calculate 'a' parameter

a = Propagation Delay / Transmission Delay = 16.7 / 60 = 0.28

Step 2: Calculate utilization

Utilization = min(1, 10 / (1 + 2×0.28)) = min(1, 10 / 1.56) = min(1, 6.41)

Result: 100% utilization

Answer: Both options achieve 100% link utilization because window size (N=10) is sufficient for the round-trip time.

Task d) If the wireless connection experiences 2% frame loss, estimate the effective throughput considering retransmissions

Frame Loss Impact: How packet loss affects actual throughput due to retransmissions

Formula:

Effective Throughput = Theoretical Throughput × (1 - Frame Loss Rate)

Wireless with 2% Frame Loss

Given:

Calculation:
